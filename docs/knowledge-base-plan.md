## 四、文档规范

### 命名规范
```
[层级]-[分类]-[主题]-[版本]-[日期]
示例：L3-乐门-活动策划流程-V1.0-20260628
```

### 模板规范
所有文档需包含：
1. 文档信息（标题、作者、日期、版本、适用范围）
2. 目录
3. 正文（结构化内容）
4. 相关资源（链接、附件）
5. 更新记录

### 权限管理
| 层级 | 可见范围 | 编辑权限 |
|------|----------|----------|
| 第一层 文明总纲 | 所有人 | 长老议会 |
| 第二层 制度规范 | 村民及以上 | 执事及以上 |
| 第三层 运营体系 | 所有人 | 执事及以上 |
| 第四层 成长体系 | 所有人 | 执事及以上 |
| 第五层 资源工具 | 所有人 | 所有人（需审核） |

## 五、与Skill共创库的联动

知识库与Skill共创库形成互补关系：

- **Skill共创库**：提供可安装的AI工作流，降低执行门槛
- **知识库**：提供详细的方法论、案例和参考资料，支撑Skill的有效使用

建议在每个Skill的references目录中引用知识库中的相关文档，形成"Skill执行+知识库参考"的完整工作流。
