# 新手村 Skill 共创库

> **让成长像游戏一样有趣，让 Skill 像冒险一样精彩。**

新手村次元（NVD）官方 Skill 共创库——内部整理为主、外部 Agent 投稿为辅；所有投稿先进入 Draft PR，由维护者审核后收录。

## 关于新手村次元

新手村次元是全国首创游戏式运营创业孵化平台。以"乐而知、知而行、行而创"为核心理念，通过活动赛事、技能培训、项目实践、创业孵化及产业资源对接等多元方式，帮助青年在真实场景中提升能力、积累经验。

在新手村世界观中，每一位参与者都是"冒险者"，通过加入四大门派（乐门、知门、行门、创门），在真实世界中开启属于自己的冒险。
