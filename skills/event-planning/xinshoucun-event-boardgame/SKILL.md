# 新手村桌游活动策划

## 概述

本Skill用于策划新手村各类桌游活动。核心理念是"用游戏连接人，用推理锻炼思维"，通过桌游这一载体，为冒险者提供日常的社交场景和思维训练机会。

## 活动类型

### 1. 新手村狼人杀
- **形式**：标准局/娱乐局/新手教学局
- **人数**：9-12人标准局，可扩展至15人
- **时长**：单局30-60分钟，系列赛2-3小时
- **角色配置**：
  - 标准局：4狼+4神+4民
  - 娱乐局：加入混血儿、盗贼、白痴等
  - 新手局：减少狼人数量，增加平民
- **裁判主持**：专业主持控场，维持秩序

### 2. 新手村剧本杀
- **形式**：盒装本/城限本/独家本/自制本
- **人数**：5-8人（根据剧本）
- **时长**：3-5小时
- **类型选择**：
  - 推理本：注重逻辑和线索
  - 情感本：注重角色体验和故事
  - 机制本：注重策略和互动
  - 恐怖本：注重氛围和沉浸
- **DM要求**：熟悉剧本、控场能力强、演绎到位

### 3. 综合桌游日
- **形式**：多款桌游轮换体验
- **推荐游戏**：
  - 策略类：卡坦岛、七大奇迹、璀璨宝石
  - 聚会类：阿瓦隆、行动代号、画物语
  - 合作类：禁闭岛、瘟疫危机、灵界
  - 推理类：米勒山谷狼人、抵抗组织、血染钟楼
- **人数**：不限，分组进行
- **时长**：半天至全天

## 策划流程

1. **确定活动类型**：狼人杀/剧本杀/综合桌游
2. **预约场地**：确认空间、桌椅、灯光、隔音
3. **招募玩家**：发布活动、接受报名、确认人数
4. **准备物料**：游戏本体、道具、零食、记分板
5. **安排人员**：主持/DM、裁判、服务
6. **现场执行**：签到、分组、规则讲解、游戏进行
7. **赛后总结**：复盘、评选、反馈、合影留念

## 运营策略

### 日常局运营
- **固定时间**：每周固定时段，培养用户习惯
- **阶梯难度**：新手→进阶→高手，分层组织
- **积分体系**：参与积分、胜场积分、MVP积分
- **段位系统**：青铜→白银→黄金→钻石→王者

### 赛事运营
- **周期性赛事**：月赛、季度赛、年度总决赛
- **赛制设计**：小组赛→淘汰赛→决赛
- **奖励设置**：奖杯、证书、游戏周边、冒险币
- **直播/录播**：提升影响力和参与感

## 注意事项

- 狼人杀需准备备用玩家名单应对跳车
- 剧本杀需提前确认玩家无撞角（重复角色体验）
- 综合桌游日需安排每桌的讲解员
- 所有桌游活动需控制噪音，避免影响他人
- 建立桌游爱好者社群，保持活跃度
