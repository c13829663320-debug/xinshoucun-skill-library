# 新手村破冰活动策划

## 概述

本Skill用于策划新手村各类破冰与团队融合活动。核心理念是"让陌生人在游戏中建立连接"，通过游戏化机制降低社交门槛，帮助冒险者快速融入新手村文化。

## 适用场景

- 新成员加入时的欢迎破冰
- 跨门派/跨团队的融合活动
- 大型活动前的暖场环节
- 季度/年度团队重组后的熟悉
- 线上转线下的首次见面

## 破冰活动类型

### 1. 主题破冰行动
参考新手村经典"破冰行动"系列：
- **冒险者认证仪式**：完成趣味挑战获得"冒险者身份证"
- **门派选择测试**：通过趣味问答帮助新人找到倾向门派
- **世界任务发布**：以剧情化方式发布首个协作任务

### 2. 快速连接游戏
- **两人三句**：两人一组，3分钟内交换3个关键信息
- **寻宝拼图**：分组寻找线索拼出新手村文化关键词
- **身份盲盒**：随机抽取冒险者身份卡进行角色扮演

### 3. 团队协作挑战
- **搭建新手村**：用有限材料搭建理想村庄模型
- **冒险者公约**：共同起草团队行为准则
- **资源争夺战**：模拟新手村资源分配策略游戏

## 策划流程

1. **明确目标**：确定破冰目的（熟悉/融合/激励/文化传递）
2. **分析对象**：了解参与者背景、人数、熟悉程度
3. **选择形式**：根据场景选择线上/线下/混合模式
4. **设计环节**：按"暖场→互动→深入→收尾"结构安排
5. **准备物料**：道具、场地、音乐、奖品清单
6. **预演检查**：时间控制、风险点、应急预案
7. **执行反馈**：现场观察、事后收集反馈

## 注意事项

- 严格控制单环节时长，避免疲劳
- 预留弹性时间应对突发情况
- 关注内向参与者，设计可选参与方式
- 融入新手村世界观元素增强归属感
- 每次活动后更新"破冰游戏库"
