# 新手村户外冒险活动策划

## 概述

本Skill用于策划新手村各类户外冒险活动。核心理念是"把游戏搬到现实世界"，通过真人角色扮演、追逐竞技、团队协作等游戏化机制，让冒险者在户外环境中体验沉浸式的冒险乐趣。

## 活动类型

### 1. 真人鹅鸭杀
- **原型**：线上游戏《鹅鸭杀》线下化
- **玩法**：角色扮演+任务挑战+推理投票
- **人数**：15-30人
- **场地**：公园/校园/商圈
- **时长**：2-3小时
- **核心机制**：角色技能、任务系统、紧急会议、投票淘汰

### 2. 猫抓老鼠
- **原型**：经典追逐游戏升级
- **玩法**：定位共享+角色升级+技能卡
- **人数**：20-50人
- **场地**：公园/校园/社区
- **时长**：1.5-2小时
- **核心机制**：GPS定位、角色变身、道具技能、安全区

### 3. 撕名牌大战
- **原型**：综艺经典游戏
- **玩法**：组队竞技+策略联盟+能力卡
- **人数**：20-40人
- **场地**：操场/广场/室内体育馆
- **时长**：2-3小时
- **核心机制**：名牌保护、联盟系统、任务复活、终极对决

### 4. 校园奇遇
- **原型**：实景解谜+剧情冒险
- **玩法**：线索搜集+剧情推进+团队协作
- **人数**：10-30人（分组）
- **场地**：校园/园区/商圈
- **时长**：3-4小时
- **核心机制**：剧情任务、线索解谜、NPC互动、多结局

## 策划流程

1. **场地勘察**：测量面积、确认安全、标记危险区
2. **规则设计**：核心机制、胜负条件、时间控制
3. **角色设计**：角色能力、平衡性、互动性
4. **物料准备**：道具、标识、急救包、通讯设备
5. **人员培训**：NPC培训、裁判培训、安全员培训
6. **技术测试**：定位软件、通讯设备、音响测试
7. **预演彩排**：完整流程跑一遍，记录问题
8. **现场执行**：签到、规则讲解、分组、游戏进行、颁奖
9. **复盘总结**：收集反馈、分析数据、优化规则

## 安全规范

- 提前勘察场地，排除安全隐患
- 准备急救包，安排安全员
- 设置安全区，禁止危险行为
- 购买活动保险
- 天气预案（雨天备选方案）
- 参与者健康声明
