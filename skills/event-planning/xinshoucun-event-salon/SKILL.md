# 新手村围炉交流会与分享会策划

## 概述

本Skill用于策划新手村各类知识分享与深度交流活动，包括围炉交流会、AI探索分享会、大讲堂、特训营等品牌系列。核心理念是"在对话中碰撞思想，在分享中共同成长"。

## 活动类型

### 1. 围炉交流会
- **形式**：小型（10-20人）、圆桌、深度对话
- **特点**：氛围轻松、话题开放、人人参与
- **适用**：跨门派交流、经验分享、问题探讨
- **经典主题**：成长故事、项目复盘、行业观察

### 2. AI探索分享会
- **形式**：中型（20-50人）、主题演讲+互动
- **特点**：前沿技术、实践案例、动手体验
- **适用**：技术传播、工具教学、趋势探讨
- **经典主题**：AI工具实操、AIGC创作、智能体开发

### 3. 新手村大讲堂
- **形式**：中大型（30-100人）、嘉宾演讲
- **特点**：专业深度、嘉宾权威、内容体系化
- **适用**：专业知识传播、嘉宾邀请、系列课程
- **经典主题**：行业洞察、创业经验、专业技能

### 4. 新手村特训营
- **形式**：系列培训（多天/多周）、小班制
- **特点**：系统学习、项目实战、导师辅导
- **适用**：技能提升、项目孵化、团队培养
- **经典主题**：活动策划、AI应用、创业基础

## 策划流程

1. **确定类型与目标**：选择活动形式，明确知识传递目标
2. **设计内容框架**：主题→大纲→详细内容→互动环节
3. **邀请嘉宾/讲师**：匹配主题，确认档期，沟通内容
4. **设计互动机制**：提问、讨论、实操、反馈收集
5. **宣传推广**：海报、文案、渠道、报名管理
6. **现场执行**：签到、开场、控场、收尾
7. **内容沉淀**：纪要、回放、资料整理、二次传播

## 内容设计原则

- **开场黄金3分钟**：用一个故事/问题/数据抓住注意力
- **每20分钟一个节奏点**：切换形式（讲→问→练→论）
- **核心观点不超过3个**：少即是多，确保记忆点
- **至少1个互动环节**：让听众从"听"变成"参与"
- **结尾行动号召**：给出明确的下一步行动建议
