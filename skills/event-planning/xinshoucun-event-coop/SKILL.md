# 新手村对外合作活动策划

## 概述

本Skill用于策划新手村各类对外合作活动。核心理念是"连接外部世界，共创价值"，通过与咖啡馆、酒吧、院校、企业、商业品牌等外部伙伴的合作，为冒险者创造更多元的成长场景和资源机会。

## 合作类型

### 1. 商业空间合作
**饮盏CAFF&BAR / 山湖日咖夜酒**
- **合作模式**：场地支持、联合活动、会员互通
- **活动形式**：
  - 主题沙龙（咖啡/酒文化+冒险者分享）
  - 创业路演（白天咖啡馆→晚上酒吧）
  - 社交派对（日咖夜酒+新手村游戏）
- **价值交换**：新手村引流×场地/餐饮支持

### 2. 院校合作
**东莞职业技术学院教育与体育学院等**
- **合作模式**：课程共建、活动联办、人才输送
- **活动形式**：
  - 校园宣讲会（新手村文化+成长路径）
  - 技能工作坊（AI/设计/策划实训）
  - 创业比赛（联合举办/协办）
- **价值交换**：教育资源×实践平台

### 3. 企业团队合建
**莞客松团队等**
- **合作模式**：团队共建、项目合作、资源共享
- **活动形式**：
  - 联合团建（新手村游戏化团建方案）
  - 黑客松/共创营（48小时极限挑战）
  - 行业交流会（技术/创业/设计方向）
- **价值交换**：组织能力×品牌曝光×人才池

### 4. 品牌联名活动
- **合作模式**：品牌赞助、联合推广、资源置换
- **活动形式**：
  - 联名周边（新手村×品牌定制）
  - 主题快闪（品牌理念+冒险者精神）
  - 线上挑战赛（社交媒体联动）

## 合作洽谈流程

1. **目标筛选**：明确合作目的，筛选潜在伙伴
2. **价值分析**：分析双方资源、需求、契合度
3. **初次接触**：发送合作意向，预约面谈
4. **方案共创**：共同设计活动方案和价值交换
5. **协议签订**：明确权责、预算、分工、知识产权
6. **联合执行**：双方团队协同推进
7. **效果评估**：数据复盘、关系维护、二次合作

## 注意事项

- 合作前务必明确知识产权归属
- 所有对外合作需经长老议会审批
- 合作协议需法律顾问审核
- 重视合作后的数据复盘和关系维护
- 建立合作方资源库，长期经营
- 每次合作都是新手村品牌展示的机会
