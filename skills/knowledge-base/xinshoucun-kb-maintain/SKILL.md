# 新手村知识库维护与管理

## 概述

本Skill用于新手村知识库的规划、建设和日常维护。核心理念是"让知识触手可及，让经验得以传承"，通过系统化的知识管理，让每位冒险者都能快速找到所需信息，让新手村的智慧不断积累。

## 知识库架构

### 第一层：文明总纲
- 新手村概述
- 世界观体系
- 冒险者文明白皮书
- 核心理念与价值观

### 第二层：运营体系
- 活动策划手册
- 社群运营指南
- 宣传推广模板
- 对外合作规范

### 第三层：成长体系
- 门派知识体系
- 技能学习路径
- 项目案例库
- 导师资源库

### 第四层：制度规范
- 组织架构
- 执事工作手册
- 财务管理制度
- 知识产权规范

### 第五层：资源库
- 设计素材
- 文案模板
- 合同模板
- 工具推荐

## 文档分类标准

### 按类型分类
| 类型 | 说明 | 示例 |
|------|------|------|
| 制度 | 规则与规范 | 执事手册、财务制度 |
| 指南 | 操作说明 | 活动策划指南、社群运营SOP |
| 案例 | 实战经验 | 项目复盘、活动回顾 |
| 模板 | 可直接使用 | 合同模板、海报模板 |
| 知识 | 学习资料 | AI教程、行业报告 |
| 数据 | 统计记录 | 活动数据、用户数据 |

### 按门派分类
| 门派 | 内容方向 |
|------|----------|
| 乐门 | 活动策划、游戏设计、体验设计 |
| 知门 | 研究报告、技术文档、学习笔记 |
| 行门 | 项目文档、设计稿、代码仓库 |
| 创门 | 商业计划、融资材料、合作协议 |

### 按权限分类
| 权限 | 可见范围 |
|------|----------|
| 公开 | 所有人可见 |
| 内部 | 村民及以上 |
| 执事 | 执事及以上 |
| 长老 | 长老及以上 |

## 标签体系

### 必打标签
- **文档类型**：制度/指南/案例/模板/知识/数据
- **所属门派**：乐门/知门/行门/创门/通用
- **权限等级**：公开/内部/执事/长老
- **状态**：草稿/审核中/已发布/已归档

### 可选标签
- **主题关键词**：AI、创业、活动、设计、运营
- **适用场景**：新人入门、活动策划、项目管理
- **更新频率**：实时/每周/每月/每年
- **重要程度**：核心/重要/一般/参考

## 文档规范

### 命名规范
```
[类型]-[门派]-[主题]-[版本]-[日期]
示例：指南-乐门-活动策划流程-V1.0-20260628
```

### 模板规范
- 封面：标题、作者、日期、版本
- 目录：自动生成的可点击目录
- 正文：结构化内容，使用标题层级
- 附录：相关资源、参考链接、更新记录

### 审核流程
1. **撰写**：作者完成初稿
2. **自审**：作者检查格式、链接、准确性
3. **同行审**：相关领域冒险者审核内容
4. **终审**：长老或指定负责人批准发布
5. **发布**：添加到知识库对应位置
6. **归档**：过时文档标记归档，保留历史版本

## 检索优化

### 目录索引
- 每个大类维护一个索引文档
- 索引包含：文档名、简介、最后更新、负责人

### 搜索关键词
- 文档标题包含核心关键词
- 文档摘要包含相关同义词
- 标签覆盖常见搜索词

### FAQ汇总
- 收集常见问题
- 每个问题链接到对应文档
- 定期更新FAQ

## 维护SOP

### 每周
- 检查新上传文档，确认分类和标签
- 更新索引文档
- 收集用户反馈的文档问题

### 每月
- 审核待发布文档
- 检查链接有效性
- 统计高频搜索词，优化标签

### 每季度
- 全面盘点知识库结构
- 归档过时文档
- 评估新增分类需求
- 用户满意度调研

### 每年
- 知识库架构review
- 权限体系review
- 历史版本清理
- 知识库年度报告

## 注意事项

- 知识库是公共资产，需共同维护
- 文档更新需注明变更内容
- 重要文档变更需通知相关方
- 定期备份，防止数据丢失
- 鼓励冒险者贡献知识，建立贡献激励机制
